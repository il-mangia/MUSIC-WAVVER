How to install Music Wavver:

    1) Open terminal in the current folder
    2) Run chmod +x install.sh
    3) Run ./install.sh (for dependencies is required SUDO)

How to uninstall Music Wavver:
    1) Open terminal in the current folder
    2) Run chmod +x uninstall.sh
    3) Run ./uninstall.sh (for dependencies is required SUDO)

How to update Music Wavver:
    1) Open terminal in the current folder
    2) Run ./install.sh
    If update doesn't work try:
    1) Run ./uninstall.sh
    2) Run ./install.sh

How to use Music Wavver without installation:
    1) Open terminal in the current folder
    2) Run chmod +x MUSIC\ WAVVER
    3) Run ./MUSIC\ WAVVER
